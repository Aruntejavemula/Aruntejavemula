  ### 🤖 [Personal Dashboard](https://github.com/Aruntejavemula/personal-dashboard)

  Self-hosted personal dashboard PWA with an **agentic AI assistant** built on AWS Bedrock (Claude Sonnet 4.5), with BYOK support for Anthropic, OpenAI, and Gemini.

  **Highlights:**
  - 🧠 Multi-provider AI dispatch with provider-agnostic agent loop
  - 📝 Notion-style multi-note editor
  - 📬 Webhook inbox for external apps (Shortcuts, Zapier)
  - 🔐 Single-user auth with signed session cookies
  - 📦 pnpm monorepo · TypeScript · Express 5 · Vite · PWA
